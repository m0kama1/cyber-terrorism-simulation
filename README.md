# Cyber Terrorism Crisis Simulator

Decision-based educational simulation for training security and intelligence professionals on managing cyber-terrorism crises through strategic choices and trade-offs.

> **Focus:** governance, coordination, public communication, and political trade-offs  
> **Not focus:** technical attack steps or operational guidance

---

## Overview

**Cyber Terrorism Crisis Simulator** is a lightweight web-based simulation (hosted via GitHub Pages) that places the learner in the role of a **national cyber crisis management cell**.

Learners make time-pressured decisions across multiple phases of a crisis and observe how choices impact key outcome indicators such as:
- Security control
- Public trust
- Continuity of critical services
- Domestic & international pressure

The simulator is designed for:
- Classroom instruction
- Executive training
- Scenario-based discussions
- AI-augmented learning (future roadmap)

## How it Works

The simulation is **decision-centric**:
1. Read the situation brief for the current phase
2. Choose one of the available decisions
3. Review the outcome explanation
4. Proceed to the next phase

No technical exploit details are included.
